本計畫依照經濟部技術處委辦計畫要點，成立苗栗產業創新推動辦公室，籌組跨組織、跨領域、跨產業之專業技術輔導團隊，推動微型創新輔導、苗栗特色產業聚落形塑、研發聯盟輔導推動等相關業務；透過產業輔導團隊連結產學、研能量，對苗栗地區在地特色產業進行輔導，針對苗栗在地企業之需求均能依照個別企業規模、產業別或技術需求能夠隨機因應，設置單一服務窗口，提供整合性客製化服務，協助企業穩健發展，並循序漸進輔導業者投入研發創新，達到產業創新與升級轉型之目標，本分項計畫包括微型關懷輔導與苗栗特色產業聚落形塑等系列性工作項目，分別說明如下。

 

微型關懷輔導機制

 

 

本計畫將透過引進學校與研發法人之輔導能量，首先針對苗栗地區之粉末冶金產業、陶瓷生活創意產業、智慧觀光、地方特色產業、農特產業品產業與文化創意產業在產品開發、生產改善、包裝設計、科技美學以、自動化、市場行銷、ICT科技能量導入及節能減碳等方面進行關懷診斷與輔導，並協助其進行微型計畫提案，協助企業能夠取得政府資源。

 

 

首先透過電話與業者洽談，了解廠商經營現況，針對廠商之需求，由苗栗產業創新推動辦公室進行訪視，釐清業者遭遇問題面向，並提出初步解決方向及可利用之政府輔導資源計畫，如廠商有明確需求且願意投入改善，再由辦公室遴派合適專家前往診斷，針對單一特定問題進行深入探討，提出解決方案。

 

 

之後透過企業創新審查機制，遴選合適對象提撥苗創微型輔導經費，協助其朝特色化、高值化、市場化與品牌化發展。此外，苗栗縣潛在重點微型企業、具產品開發企圖心之企業與企業形象發展決心之企業亦為潛在輔導對象。透過上述做法，將可促進企業發展，穩固地方特色產業，進而帶動地方產業經濟發展，其概念如下所示。
